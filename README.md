# Sales-Performance-Tracker

- Designed a multi-sheet Excel workbook to log and analyze 400+ sales transactions across 10 reps, 4 regions, and 6 products, with dynamic formulas, KPI summaries, and conditional formatting for real-time performance tracking.
- Built interactive Power BI reports featuring KPI cards, monthly trend charts, regional maps, and target vs actual comparisons using custom DAX measures.
- Enabled data-driven decision-making by visualizing quarterly revenue trends, rep-level achievement rates, and product performance across a full fiscal year.
